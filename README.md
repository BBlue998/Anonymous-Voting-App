# 익명 투표 서비스 데이터 분석 및 UX 개선 보고서

**분석 기간:** 2026-01-27 ~ 2026-02-27  
**팀원:** 김선구, 노유림, 박인화, 정지영  
**사용 도구:** Python, MySQL

---

## 프로젝트 개요

익명 투표 기반 모바일 서비스의 운영 DB를 분석하여, 서비스가 이탈한 원인을 데이터로 규명하고 UX 구조 개선 방향을 제시한 프로젝트입니다.

---

## 주요 분석 내용

- **사용자 분포 및 투표 퍼널 분석** — 전체 67만 명 가입자의 행동 흐름 추적 (투표 → 알림 읽음 → 상세 열람)
- **핵심 지표 분석** — WAU/MAU, MRR, 유지율/재구매율 시계열 분석
- **서비스 전환점 탐지** — 2023년 5월 28일을 기점으로 투표 참여율 30% 이상 급감 확인
- **K-Means 클러스터링** — 정책 변화 전후 사용자 행동 변화량(Delta) 기반으로 3개 유형 분류
  - Cluster 0: 형식적 잔존형 (방문은 유지, 실질 참여 붕괴)
  - Cluster 1: 핵심 유지·확장형 (정책 적응 후 활동 유지)
  - Cluster 2: 전통적 이탈형 (방문·참여 모두 감소)

---

## 핵심 발견

- 포인트 보상 제도가 앱 내 활동 증가로 이어지지 않음 (상관관계 無)
- 알림 도달률은 높으나(55.5%) 상세 열람 전환율은 8.97%에 불과
- 상위 10% 유저가 전체 선택의 51.4%를 차지하는 편향 구조 확인

---

## UX 개선 제안

서비스를 **"친구 기반 SNS"에서 "공동체 경험 플랫폼"** 으로 재정의

- 학급/그룹 단위 폐쇄형 그룹 구조 도입
- Daily 5-Set 질문 구조 + 인물 투표 1회 제한으로 편향 완화
- 개별 알림 → 궁금증 유발형 요약 알림으로 전환
- 포인트 사용처를 락커 커스터마이징으로 확장
